> 키네마틱 시퀀스에서 피팅 변수와 가장 직접 연결되는 신체 분절.

손목 세트 각도(샤프트와 팔의 각도)가 다운스윙에서 어떻게 변하는가를 봅니다. 이 패턴이 EI 선택과 CPM 결정의 직접적 근거가 됩니다.

| 손목 고리 상태 | 피팅 처방 |
|---|---|
| 정상 릴리즈 | 표준 EI / 현재 CPM |
| 얼리 릴리즈 (캐스팅) | Tip Stiff / CPM +5~10 / 무게 ↑ |
| 레이트 릴리즈 | Tip Soft / CPM −5~10 / 무게 ↓ |
| 손목 고리 불안정 | 그립 두께·길이 재확인 |
