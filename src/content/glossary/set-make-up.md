> 14개 클럽의 구성.

R&A·USGA 규정상 라운드에 휴대 가능한 클럽은 최대 14개. 어떤 14개로 채울지 = 세트 메이크업.

**일반 구성:**
- 드라이버 (1)
- 페어웨이 우드 / 하이브리드 (2~3)
- 아이언 (5~7)
- 웨지 (3~4: PW · GW · SW · LW)
- 퍼터 (1)

**핸디캡별 권장 구성:**
- 보기 골퍼 이하: 드라이버 + 우드 2 + 하이브리드 1 + 아이언 5번~PW + 웨지 3 + 퍼터
- 5핸디캡 이하: 드라이버 + 우드 1 + 하이브리드 1 + 아이언 4번~PW + 웨지 4 + 퍼터
- 시니어·여성: 롱아이언(3·4번) → 하이브리드 대체 강력 권장

**세트 메이크업 검증 항목 (PART 11):**
- 클럽별 거리 갭 10~15야드 균일
- 로프트 간격 클럽당 4° 내외
- 길이 흐름 클럽당 0.5인치
- SW 흐름 일관 (또는 진행)
- CPM FM 흐름 클럽당 5~8 균일 증가
- 총중량 점진 증가 (드라이버 → 웨지)

→ 거리 갭 / 로프트 갭 항목 참조
