> 임팩트 직전, 골퍼가 클럽을 잡고 공 앞에 서는 자세.

피팅 인터뷰에서 가장 자주 듣는 단어 중 하나. "셋업이 안 잡힌다", "셋업에서 그립이 답답하다", "셋업하면 페이스가 닫혀 보인다" 등.

**셋업과 피팅 변수의 연결:**

| 셋업 호소 | 의심 변수 |
|---|---|
| "그립이 답답함" | 그립 사이즈 (큼) |
| "그립이 헐렁함" | 그립 사이즈 (작음) |
| "어드레스 시 페이스 닫힘" | 슬리브 설정 / 페이스 프로그레션 |
| "어드레스 시 페이스 열림" | 슬리브 설정 / 호젤 휨 작업 |
| "토 끝이 들림" | 라이각 플랫 → 업라이트 검토 |
| "힐 끝이 들림" | 라이각 업라이트 → 플랫 검토 |
| "공이 너무 멀게 느껴짐" | 길이 + 라이각 + 신장 매칭 |

**셋업은 정적 라이각의 기준점**이지만 **임팩트는 다이나믹 라이각**입니다. 둘은 다릅니다 — 다이나믹 라이 테스트로 보정합니다.
