> Dynamic Loft − Attack Angle. 스핀량을 결정하는 핵심 변수.

쉽게: 페이스가 위로 향한 각도와 클럽이 위/아래로 움직이는 각도의 차이. 이 차이가 클수록 스핀이 많이 발생합니다.

투어/핸디캡별 평균 (드라이버):

| 구분 | Spin Loft |
|---|---|
| PGA 투어 | 14.7° |
| LPGA 투어 | 15.0° |
| 5핸디 (남) | 15.8° |
| 10핸디 (남) | 17.3° |
| 일반 골퍼 (14.5핸디) | 18.3° |
| 보기 골퍼 | 18.2° |

**Spin Loft가 크면 → Smash Factor 낮음 → 거리 손실**
**Spin Loft가 작으면 → Smash Factor 높음 → 거리 증가**

Spin Loft를 줄이려면 디로프트(볼을 눌러치기)를 잘해야 합니다. **PGA 투어가 일반 골퍼보다 4° 낮은 이유.**

📊 트랙맨 데이터 (Club) / 연관: Dynamic Loft, Attack Angle, Smash Factor, Spin Rate
