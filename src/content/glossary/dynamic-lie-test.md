> 실제 스윙 시 클럽이 지면과 만나는 각도를 측정하는 현장 테스트.

테스트 방법: 솔에 라이 테이프를 붙이고 임팩트 보드 위에서 타격. 자국 위치로 라이각 편차를 읽어냅니다.

- 자국이 토 쪽: 라이각이 플랫. 우측 미스 경향 → 업라이트로 1° 조정 검토
- 자국이 힐 쪽: 라이각이 업라이트. 좌측 미스 경향 → 플랫으로 1° 조정 검토

클럽메이커스 공작소: **테스트 70,000원 / 세트 교정 포함 150,000원, 3회 무료 사후관리.**

→ 라이각 조정 작업 / 그루브라인 / 솔 항목 참조
