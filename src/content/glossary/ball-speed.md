> 임팩트 직후의 볼 속도.

비거리를 결정하는 가장 직접적인 지표. **드라이버는 1mph = 약 2야드.**

투어/핸디캡별 평균 (드라이버):

| 구분 | 볼 스피드 |
|---|---|
| PGA 투어 | 167 mph |
| LPGA 투어 | 139 mph |
| 5핸디 (남) | 147 mph |
| 10핸디 (남) | 138 mph |
| 일반 골퍼 (14.5핸디) | 133 mph |
| 보기 골퍼 | 131 mph |
| 5핸디 (여) | 125 mph |

볼 스피드는 클럽 스피드와 임팩트 효율(Smash Factor)이 결합된 결과입니다. 클럽 스피드만 올리고 타점이 빗나가면 볼 스피드는 오히려 떨어집니다.
