# 클럽메이커스 용어사전 (Glossary)

> **이 사전이 다른 사전과 다른 이유**
>
> 일반 사전은 부품을 정의합니다. 이 사전은 **부품이 작업대 위에서 어떻게 다뤄지는지**를 함께 보여줍니다.
>
> 예를 들어 그루브라인은 단순한 페이스의 홈이 아닙니다. 라이각 조정 작업에서 헤드를 밴딩기에 정렬하는 좌표축입니다. 책 『실제적인 클럽피팅』의 판단 기준과 작업대 위의 실제 흐름이 그대로 들어가 있습니다.
>
> 골퍼와 피팅 전문가 양쪽이 모두 읽을 수 있도록, 첫 한 줄은 누구나 이해할 수 있게, 다음 단락은 깊이 있게 정리했습니다.

### 거리 갭 (Distance Gap)
*Distance Gap*

> 클럽 사이 비거리 차이.

세트 메이크업의 핵심 지표. 일반적으로 **클럽 사이 10~15야드** 간격이 이상적입니다. 갭이 너무 크면 어프로치 시 풀스윙·하프스윙 사이에서 거리 조절이 어려워집니다.

흔한 실수:
- PW(45°) → 50° 웨지: 5° 차이지만 거리는 약 12야드 (적절)
- PW(43° 스트롱 로프트) → 50° 웨지: 7° 차이로 거리 갭 18야드 이상 (큰 갭)

스트롱 로프트 아이언을 쓰는 골퍼는 대부분 **갭 웨지(GW)를 추가**해서 메꿉니다. 클럽메이커스 풀피팅에서 거리 갭은 마지막 단계에 반드시 검증합니다.

### 골퍼의 조건
*Golfer's Conditions*

> 피팅 전에 반드시 기록해야 하는 골퍼의 11가지 변수.

신체·근력·피로/컨디션·헤드스피드·템포/리듬·전환 강도·릴리즈 타이밍·타점 경향·구질 경향·의도/보상·반복성. 이 11가지가 모이면 골퍼의 "구조"가 보입니다. 같은 헤드스피드 95mph라도 편차가 ±2mph인 골퍼와 ±8mph인 골퍼는 완전히 다른 처방을 받아야 합니다. **숫자는 결과이고, 조건이 원인입니다.**

### 그라인드 (Grind)
*Wedge Grind*

> 웨지 솔의 형태. 라이·디봇 깊이·스윙 타입에 맞춤.

같은 로프트·바운스라도 그라인드가 다르면 잔디 위에서의 거동이 달라집니다.

| 그라인드 | 솔 형태 | 적합 |
|---|---|---|
| **F (Full)** | 솔 전체 사용 | 풀스윙 위주 |
| **M (Mid)** | 중간 폭 | 만능형 |
| **S (Standard)** | 표준 | 일반 |
| **D (Wide)** | 와이드 솔 | 부드러운 라이·시니어 |
| **K (Bunker)** | 큰 바운스 | 모래·푹신한 라이 |
| **L (Low)** | 트레일링 그라인드 | 오픈 페이스 샷 |
| **T (Tour)** | 힐·토 깎임 | 상급자 매뉴팩처 |

대부분의 메이커가 자체 표기를 씁니다 (Vokey SM10의 F·S·M·D·K·L·T 등). 본질은 **솔이 잔디에 어떻게 안착하는가**입니다.

### 그라파이트 / 그래파이트 (Graphite)
*Graphite Shaft*

> 카본 파이버로 만든 샤프트. "카본 샤프트"라고도 부릅니다.

스틸 샤프트보다 가볍고, 토크가 크며, 진동 흡수가 좋습니다. 시니어·여성·관절 통증이 있는 골퍼에게 권장.

| 항목 | 그라파이트 | 스틸 |
|---|---|---|
| 무게 | 40~85g | 95~130g |
| 토크 | 2.5~6.0° | 1.5~3.0° |
| 진동 | 적음 | 큼 |
| 가격 | 비쌈 | 저렴 |
| 일관성 | 중간 (제조 편차 있음) | 매우 일정 |

⚠️ **드라이버는 거의 항상 그라파이트**입니다. 아이언은 헤드스피드 80mph 이상이면 스틸도 가능, 그 이하면 그라파이트 우선.

표기 주의: 한국에서는 "그라파이트"와 "그래파이트" 두 표기가 혼용됩니다. 영문 graphite의 한글 표기 차이일 뿐 같은 것입니다.

### 그루브라인 (Score Line / Groove Line) 🛠️
*Score Line / Groove*

> 페이스 표면의 가로 홈. "스코어라인"이라고도 부릅니다.

스핀을 만들고 잔디·물기를 흘려보내는 기능을 합니다. 그러나 클럽메이커스 작업에서 그루브라인은 또 다른 역할이 있습니다.

**🔧 그루브라인과 라이각 조정 작업의 연결**

라이각을 조정할 때는 헤드를 **밴딩기(Bending Machine)에 고정**합니다. 이때 기준이 되는 것이 그루브라인입니다.

1. 헤드를 밴딩기에 물림
2. **그루브라인이 지면과 정확히 평행**이 되도록 정렬
3. 호젤이 지면과 이루는 각도 = 현재 라이각
4. 호젤을 위/아래로 휘어서 라이각 변경
5. 다시 그루브라인 평행 확인

그루브라인이 평행하지 않은 상태에서 호젤을 휘면, 측정 라이각과 실제 라이각이 어긋납니다. **그루브라인은 라이각 작업의 좌표축**입니다.

⚠️ 머슬백·블레이드는 그루브라인이 명확해서 정렬이 쉬운 반면, 일부 캐비티백은 페이스 디자인이 비대칭이라 정렬에 더 신경 써야 합니다.

→ 라이각 조정 작업 / 호젤 / 다이나믹 라이 테스트 항목 참조




**→ 공작소 작업: [라이 · 로프트](/workshop#lie-loft)**

### 그립 (Grip)
*Grip*

> 클럽의 손잡이 부분. 그러나 단순한 부속이 아닙니다.

그립은 골퍼가 클럽과 직접 접촉하는 유일한 지점입니다. 그립 사이즈가 0.5mm만 달라져도 손목 액션이 달라지고, 손목 액션이 달라지면 페이스 컨트롤이 달라집니다. 손이 작은 골퍼에게 작은 그립을 끼우면 스윙웨이트가 올라가지만, 그것은 그립 사이즈가 1순위인 골퍼에게 받아들여야 할 거래입니다. **그립은 후순위가 아닙니다. 사람마다 순위가 다릅니다.**

손 길이 측정 → 그립 사이즈 결정:
- 17cm 미만: 언더사이즈/주니어
- 17.0~19.0cm: 스탠다드
- 19.1~20.5cm: 미드사이즈 (+2겹 빌드업)
- 20.6~23.0cm: 오버사이즈 (+4겹 빌드업)

→ 리그립 / 그립 솔벤트 항목 참조

### 그립 솔벤트 / 그립 테이프 🛠️
*Grip Solvent / Grip Tape*

> 그립 교체 작업의 두 핵심 소모재.

- **그립 테이프**: 양면테이프. 보통 2인치 폭. 샤프트 둘레에 한 겹 감음
- **그립 솔벤트**: 미네랄 스피릿 또는 전용 솔벤트. 테이프를 미끄럽게 만들어 그립을 슬라이드 장착

작업 중 솔벤트가 마르기 전에 그립을 끝까지 밀어 넣고 정렬해야 합니다. **30초가 골든 타임**입니다. 정렬이 틀어지면 다시 떼어내고 처음부터 해야 합니다.

### 길이 (Length)
*Club Length*

> 클럽 전체의 길이. 피팅에서 가장 먼저 결정해야 하는 변수.

길이는 단순한 편의가 아니라 **타점이 형성되는 구조 자체**를 결정합니다. 길이가 맞지 않으면 타점이 일정하지 않고, 타점이 일정하지 않으면 방향도 무게도 강성도 평가할 수 없습니다.

- 길이 +0.5인치 → 타점이 토 쪽으로, 스윙웨이트 +3
- 길이 -0.5인치 → 타점이 힐 쪽으로, 스윙웨이트 -3

신장 기준 드라이버 출발 길이:
- 165cm 이하: 44.5"
- 165~175cm: 45.0"
- 175~185cm: 45.5"
- 185cm 이상: 45.75"

PGA 투어 평균 드라이버 길이는 약 44.5"로, 시중 카탈로그의 "표준" 45~46"보다 짧습니다. **타점 일관성이 길이보다 우선합니다.**

### 낙하각 (Landing Angle) 📊
*Descent Angle / Landing Angle*

> 공이 지면에 닿을 때의 입사 각도.

탄도의 "끝"을 보는 지표입니다. 낙하각이 너무 낮으면 공이 굴러서 그린에 멈추지 않습니다. 너무 높으면 캐리 거리가 짧아집니다.

투어 평균:
- 드라이버: PGA 38° / LPGA 37°
- 6번 아이언: PGA 50° / LPGA 46°

어프로치 샷에서는 45° 이상 권장. 낙하각 1° 차이가 1.5~2야드 구름 차이를 만듭니다. 평균 비거리만 보면 놓치는 것이 낙하각입니다. **거리만큼 중요한 것은 어떻게 떨어지는가입니다.**

### 납 테이프 (Lead Tape) 🛠️
*Lead Tape*

> 헤드 또는 샤프트에 부착해 무게를 미세 조정하는 납 재질 테이프.

가장 단순하고 빠른 SW·총중량 보정 도구입니다. 1g 단위 미세 조정 가능.

**부착 위치별 효과:**
- 헤드 솔 후방 → SW 증가, 탄도 약간 상승
- 헤드 솔 토 → 토 무게, 페이스 열림 경향 (드로 보정)
- 헤드 솔 힐 → 힐 무게, 페이스 닫힘 경향 (페이드 보정)
- 그립 아래 (샤프트 버트) → 카운터밸런스 효과, SW 감소

**현장 사례 (작업시트 발췌):** *"9번·PW은 헤드가 가벼워 최대 8g 무게추를 넣고도 2g 정도의 납테이프를 붙여서 맞추었습니다."* — 무게추 한계를 납 테이프로 보완하는 실제 작업.

⚠️ 납 테이프는 임시 조정용이 아니라 **합법적 영구 보정 도구**입니다. R&A·USGA 모두 허용. 단, 라운드 중에는 부착 위치를 변경할 수 없습니다.

→ 무게추 / 카운터밸런스 / SW 항목 참조

### 다이나믹 라이 (Dynamic Lie) 📊
*Dynamic Lie*

> 임팩트 순간 샤프트와 지면 사이의 실제 각도.

정적 라이각(가만히 세워둔 각도)과 다이나믹 라이각(스윙 임팩트 순간의 각도)은 다릅니다. 임팩트 순간 샤프트가 휘면서 헤드 토 쪽이 들리거나 힐 쪽이 들립니다. 트랙맨 등 론치모니터에서 직접 측정하지만, 클럽메이커스는 임팩트 보드 + 라이 테이프 방식으로도 정밀 측정합니다.

### 다이나믹 라이 테스트 🛠️
*Dynamic Lie Test*

> 실제 스윙 시 클럽이 지면과 만나는 각도를 측정하는 현장 테스트.

테스트 방법: 솔에 라이 테이프를 붙이고 임팩트 보드 위에서 타격. 자국 위치로 라이각 편차를 읽어냅니다.

- 자국이 토 쪽: 라이각이 플랫. 우측 미스 경향 → 업라이트로 1° 조정 검토
- 자국이 힐 쪽: 라이각이 업라이트. 좌측 미스 경향 → 플랫으로 1° 조정 검토

클럽메이커스 공작소: **테스트 70,000원 / 세트 교정 포함 150,000원, 3회 무료 사후관리.**

→ 라이각 조정 작업 / 그루브라인 / 솔 항목 참조




**→ 공작소 작업: [라이 · 로프트](/workshop#lie-loft)**

### 디플렉션 (Deflection)
*Shaft Deflection*

> 샤프트가 외력을 받았을 때 휘어지는 정도와 형태.

샤프트가 어디서 얼마나 휘는가를 보는 지표입니다. 같은 강성의 샤프트라도 휘는 위치가 다르면 임팩트 타이밍이 달라집니다. EI 프로파일과 함께 측정하며, 킥포인트 분류의 근거가 됩니다.

→ 카탈로그의 "하이킥/미드킥/로우킥" 표기는 이 디플렉션 데이터를 단순화한 것입니다. **현장에서는 표기보다 실측을 신뢰합니다.**

### Dynamic Loft (다이나믹 로프트) 📊
*Dynamic Loft*

> 임팩트 순간 클럽 페이스의 실제 로프트.

표기 로프트가 10.5°라도, 임팩트 순간 골퍼의 Attack Angle과 샤프트 휨에 따라 Dynamic Loft는 12°가 될 수도, 9°가 될 수도 있습니다. **발사각의 75~80%가 Dynamic Loft로 결정됩니다.**

투어/핸디캡별 평균:

| 구분 | 드라이버 | 6번 아이언 |
|---|---|---|
| PGA 투어 | 12.8° | 20.2° |
| LPGA 투어 | 15.5° | 23.6° |
| 5핸디캡 (남) | 13.2° | — |
| 10핸디캡 (남) | 14.1° | — |
| 일반 골퍼 (14.5핸디) | 15.1° | — |
| 보기 골퍼 | 14.3° | — |

Dynamic Loft가 너무 낮으면 탄도가 부족하고 런이 많아져 거리 조절이 어려워집니다. **디로프트(볼을 눌러치기)가 잘 되는지의 지표.**

### 라이 앵글 (Lie Angle)
*Lie Angle*

> 클럽이 지면에 놓였을 때 샤프트와 지면 사이의 각도.

라이각은 좌우 방향만 보정하는 변수가 아닙니다. **타점 위치와 출발 방향을 동시에 바꾸는 구조 변수**입니다. 라이각이 1° 업라이트면 출발 방향이 약 4야드 좌측으로 이동합니다.

기준값 (아이언):
- 4번: 60.5°
- 7번: 62.0~62.5°
- PW: 64.0°
- SW: 64.0~64.5°

길이가 확정된 이후에만 라이각 조정에 의미가 있습니다. **순서를 바꾸면 안 됩니다.**

→ 라이각 조정 작업 / 다이나믹 라이 테스트 / 그루브라인 항목 참조

### 라이각 조정 작업 (Lie Angle Adjustment / Bending) 🛠️
*Lie Angle Bending*

> 호젤을 미세하게 휘어 라이각을 변경하는 작업.

클럽메이커스에서 가장 자주 의뢰받는 작업 중 하나. **모든 라이각 조정은 호젤 휨**입니다.

**작업 흐름:**

1. **다이나믹 라이 테스트** (라이 테이프 + 임팩트 보드)로 현재 편차 측정
2. 헤드를 **밴딩기**에 고정 — **그루브라인이 지면과 평행**
3. 정적 라이각 측정 (62.0° 등)
4. 호젤을 위/아래로 휘어 ±1~2° 조정
5. 재측정 → 평행 확인

**한계:**
- **포지드(연철) 헤드**: ±2° 조정 가능, 안전
- **캐스팅(주조) 헤드**: ±1° 권장, 그 이상은 헤드 균열 위험
- **단조 합금**: 메이커별 권장값 확인 필수

⚠️ 호젤은 **반복적으로 휘면 약해집니다.** 잘못된 방향으로 휘었다가 다시 되돌리는 작업은 호젤 수명을 단축시킵니다. 첫 작업에서 정확한 각도로 휘는 것이 중요합니다.

클럽메이커스 공임표: **라이/로프트 조정 10,000원** / 다이나믹 라이 테스트 + 세트 교정 150,000원 (3회 무료 사후관리)

→ 라이 앵글 / 그루브라인 / 호젤 / 다이나믹 라이 테스트 항목 참조




**→ 공작소 작업: [라이 · 로프트](/workshop#lie-loft)**

### 레이트 릴리즈 (Late Release)
*Late Release / 라그 과다*

> 임팩트 이후에도 손목 세트 각도가 크게 남아 있는 패턴.

### 로프트 (Loft)
*Loft Angle*

> 클럽 페이스가 위쪽을 향한 각도.

탄도의 1차 결정 변수입니다. 그러나 표기 로프트와 Dynamic Loft는 다릅니다. **로프트만 바꿔서 탄도를 고치는 것은 가장 마지막에 시도하는 방법입니다.**

스트롱 로프트 주의: 7번 아이언 27° 같은 스트롱 로프트는 거리는 늘지만 갭이 무너지고 탄도가 부족해집니다. 시니어·여성 골퍼에게 특히 위험합니다.

### 로프트 갭 (Loft Gap)
*Loft Gap*

> 클럽 사이 로프트 차이.

세트 메이크업의 또 하나의 핵심 지표. 일반적으로 **클럽당 4° 내외**가 균일합니다.

이상적 흐름:
- 7번(31°) → 8번(35°) → 9번(39°) → PW(43°) → GW(48°) → SW(54°) → LW(58°)
- 클럽당 4° 차이 + PW·GW 사이 5° 차이 = 거리 갭 약 12야드

스트롱 로프트 주의: 7번이 27°, PW가 43°라면, PW와 GW(48°) 사이는 5°지만 7번~PW 사이는 16°에 7개 클럽이 들어 있습니다. 갭이 균일하지 않으면 거리 조절이 어려워집니다.

→ 거리 갭 / 세트 메이크업 항목 참조

### Lie Angle Balanced (라이 앵글 밸런스드) 🆕
*L.A.B. — Lie Angle Balanced*

### LS 드라이버 (Low Spin Driver) 🆕
*Low Spin Driver*

> 빠른 헤드스피드 골퍼를 위해 설계된 저스핀 드라이버 카테고리.

CG(무게중심)를 페이스 쪽 앞부분에 배치해 임팩트 시 스핀량을 낮춥니다. 결과: 더 낮고 더 멀리 뻗는 탄도(penetrating flight). 풍속에 강함.

대표 모델 (2025~2026):
- **TaylorMade Qi35 LS** — 430cc, chromium carbon + steel + aluminum + tungsten + titanium 5종 멀티 머티리얼. TAS(Trajectory Adjustment System) 13g + 3g×2 어드저스터블 무게. 페이드 바이어스
- **Ping G440 LST** — 컴팩트 헤드, 로우킥 컨피그레이션
- **Callaway Elyte** — 어드저스터블 페이스 앵글

특징:
- 헤드 크기: 430~460cc (컴팩트)
- 페이드 바이어스 또는 뉴트럴

⚠️ **모든 골퍼에게 좋은 드라이버가 아닙니다.** 헤드스피드가 충분하고(105mph+) 타점이 안정된 상급자용입니다. 중핸디캡 이하 골퍼는 LS 모델보다 Core/Max 모델에서 더 나은 결과를 얻습니다.

### 리그립 (Re-grip / 그립 교체) 🛠️
*Re-grip*

> 그립을 교체하는 작업.

클럽 작업 중 가장 흔하고, 가장 효과 빠른 단일 작업입니다. 계절 한 번 지나면 그립이 미끄러지기 시작합니다 — 그러면 손이 무의식적으로 더 세게 잡고, 손목 액션이 달라지고, 페이스가 흔들립니다. **그립 하나가 슬라이스의 원인이 될 수 있습니다.**

**작업 흐름:**
1. 기존 그립 제거 (커터 또는 압축 공기)
2. 양면테이프 부착
3. 그립 솔벤트 또는 미네랄 스피릿 도포
4. 새 그립 슬라이드 + 정렬 (30초 안에)
5. 24시간 건조

클럽메이커스 공임표: **그립 장착 5,000원 / 그립 추출 5,000원 / 퍼터 그립 7,000원** (그립 가격 별도)

💡 **권장 교체 주기:**
- 일반 라운더 (월 1~2회): 1년 1회
- 주 2~3회 라운드: 6개월 1회
- 연습 많은 골퍼: 3~4개월

→ 그립 / 그립 솔벤트 항목 참조

### 리샤프트 (Re-shaft / 샤프트 교체) 🛠️
*Re-shaft*

> 헤드는 그대로 두고 샤프트를 교체하는 작업.

새 클럽을 사기 전에 가장 먼저 검토해야 할 옵션입니다. 헤드가 마음에 들면, 샤프트만 바꿔도 완전히 다른 클럽이 됩니다.

**작업 흐름:**
1. 기존 샤프트 추출 (히트건으로 에폭시 가열 → 헤드·샤프트 분리)
2. 호젤 내부 청소 (오래된 에폭시·잔여물 제거)
3. 새 샤프트 팁 트리밍 (메이커 권장값에 맞춰)
4. 팁 표면 처리 (스틸: 연마 / 그라파이트(그래파이트): 코팅 제거)
5. 에폭시 도포 + 헤드 장착
6. 정렬 (스파인 / 로고)
7. 24시간 경화 → 길이재단 → 그립 장착

클럽메이커스 공임표:
- **샤프트 교체 18,000원** (기본)
- **길이재단 샤프트 교체 25,000원** (재단 + 트리밍 포함)
- **클럽 제작 25,000원** (헤드·샤프트·그립 모두 신품)

⚠️ 작업은 당일 불가. **에폭시 24시간 경화 시간 필요.** 빠른 경화 에폭시도 있지만 강도가 떨어집니다.

→ 에폭시 / 샤프트 분리 / 트리밍 항목 참조

### 머슬백 / 캐비티백

### 무게중심 (CG)
*Center of Gravity*

> 헤드의 무게가 모이는 점.

CG의 위치(전/후·상/하·토/힐)가 탄도와 관용성을 결정합니다.

- **Low CG + Deep CG**: 탄도 높음, 스핀 증가, 관용성 높음 (시니어·여성 권장)
- **High CG + Forward CG**: 탄도 낮음, 스핀 감소, 거리 증가 (스피드 빠른 골퍼) — LS 드라이버의 핵심 특성
- **Heel-bias CG**: 페이스 닫힘 경향 (슬라이스 보정)
- **Toe-bias CG**: 페이스 열림 경향 (훅 보정)

CG는 헤드 설계의 의도가 가장 명확하게 드러나는 지표입니다.

### 무게추 (Weight Screw) 🛠️
*Weight Screw / Sole Weight*

> 헤드 솔 또는 후방에 장착되는 나사형 무게.

어드저스터블 헤드의 핵심 부속. 무게추 위치와 무게를 바꿔서 SW·MOI·페이드/드로 바이어스를 조정합니다.

**대표 시스템:**
- **TaylorMade TAS** (Trajectory Adjustment System): 13g × 1 + 3g × 2
- **Ping**: 후방 단일 슬라이딩 무게
- **Callaway**: 토·힐 또는 후방 위치
- **PXG**: 다중 무게추 (3~4개)

**아이언 헤드 무게추:**
어드저스터블 드라이버뿐 아니라 일부 아이언도 무게추를 받습니다. **현장 사례(작업시트):** *"V300 헤드가 많이 가벼운 편입니다. 9번·PW은 헤드가 가벼워 최대 8g 무게추를 넣고도 2g 정도의 납테이프를 붙여서 맞추었습니다."*

| 위치 | 효과 |
|---|---|
| 후방 (Back) | 관용성 ↑, MOI ↑, 탄도 약간 ↑ |
| 전방 (Front) | 스핀 ↓, 거리 ↑ (LS 드라이버 특성) |
| 토 (Toe) | 페이드 바이어스 |
| 힐 (Heel) | 드로 바이어스 |

### MOI (Moment of Inertia)
*Moment of Inertia / 관성모멘트*

> 헤드(또는 클럽)가 회전에 저항하는 정도.

빗맞은 샷의 손실을 결정하는 가장 중요한 지표 중 하나.

- **헤드 MOI**: 페이스 좌우 비틀림에 대한 저항. 높을수록 빗맞은 샷의 손실 적음
- **클럽 MOI**: 클럽 전체의 회전 저항. 같은 스윙웨이트라도 MOI는 다를 수 있음

클럽메이커스는 **MOI 매칭** 서비스 제공 (25,000원). 같은 스윙웨이트의 세트라도 MOI가 다르면 후반에 리듬이 무너집니다.

**카운터밸런스**와 함께 사용하면 SW를 낮추면서 MOI를 유지할 수 있습니다.

→ MOI 매칭 / 카운터밸런스 항목 참조

### MOI 매칭 (MOI Matching) 🛠️
*MOI Matching*

> 세트 전체의 클럽 MOI를 동일하게 맞추는 작업.

SW(스윙웨이트)는 14인치 받침점 기준 정적 무게감입니다. 그러나 **MOI는 클럽 전체의 회전 저항**이며, 더 정확하게 골퍼가 느끼는 휘두름의 일관성을 보여줍니다.

같은 SW 세트라도 MOI가 클럽마다 다를 수 있습니다 → 후반에 리듬이 무너지는 원인.

**현장 사례 (작업시트):**
- *"관성모멘트 7번 기준 2,680 kg·cm²로 맞추었습니다 (오차 10 이내)"*
- *"아이언의 MOI 오차는 1g 미만으로 맞추었습니다"*
- *"최종 MOI 2,740 (오차 10 미만)"*

세트 전체를 7번 기준으로 통일하는 것이 일반적. 카운터밸런스(버트 무게) + 헤드 무게추 + 납 테이프 조합으로 매칭합니다.

클럽메이커스 공임표: **MOI 매칭 25,000원** (클럽당)

→ MOI / 카운터밸런스 / 무게추 / 납 테이프 항목 참조

### 바운스 (Bounce)
*Bounce Angle*

> 웨지 솔의 리딩 엣지와 가장 낮은 솔 지점 사이의 각도.

웨지의 핵심 변수. 바운스가 클수록 솔이 잔디·모래에 박히지 않고 미끄러집니다.

| 바운스 | 분류 | 적합한 라이 |
|---|---|---|
| **Low (4~6°)** | 로 바운스 | 단단한 라이, 깎듯이 치는 골퍼, 오픈 페이스 |
| **Mid (7~10°)** | 미드 바운스 | 만능형, 다양한 라이 |
| **High (11~14°)** | 하이 바운스 | 부드러운 라이, 푹신한 모래, 시니어 |

⚠️ **시니어·여성 골퍼에게 로 바운스 웨지는 거의 항상 잘못된 선택입니다.** 디봇이 얕고 쓸어치는 경향이 강하면 미드~하이 바운스가 안전.

클럽메이커스 공임표: **웨지 바운스 각 조정 30,000원** (당일 작업 불가)

### 벌지 & 롤 (Bulge & Roll)
*Bulge & Roll*

> 우드·드라이버 페이스의 곡률.

페이스가 평평하지 않고 미세하게 둥근 이유가 있습니다. **빗맞은 샷의 방향을 보정**하기 위해서입니다.

- **벌지 (좌우 곡률)**: 토 타점에서 좌측, 힐 타점에서 우측으로 출발. 기어이펙트로 반대 방향 곡선 → 결과적으로 중앙 복귀
- **롤 (상하 곡률)**: 상단 타점에서 발사각 증가, 하단 타점에서 감소

아이언에는 벌지와 롤이 거의 없습니다. 그래서 아이언의 빗맞음은 그대로 결과에 반영됩니다.

### 샤프트 (Shaft)

### 샤프트 스파인 (Shaft Spine) 🆕
*Shaft Spine*

> 샤프트 제조 과정에서 필연적으로 생기는 가장 강한 포인트.

카본·스틸 모두 판이 둥글게 말려 합쳐지는 부분이 있고, 그 겹쳐지는 곳이 스파인입니다. 모든 샤프트에 존재하지만, 모든 샤프트 회사가 스파인 정렬을 하지는 않습니다.

**측정 방법 3가지:**

1. **베어링** — 샤프트 양 끝(버트·팁)에 베어링 고정 후 튕기듯 돌릴 때, 멈추는 위치가 스파인
2. **FLO (Flat Line Oscillation)** — 샤프트를 클램프에 고정, 헤드부에 레이저 부착. 튕길 때 레이저가 일직선으로 움직이면 정렬됨
3. **CPM 보드** — CPM 측정 시 진동 패턴으로 정렬 확인

**정렬 방향에 따른 효과:**
- 스파인이 **3시 또는 9시 방향** → 상하 진동 안정 → 방향성 잡힘
- 스파인이 **12시 또는 6시 방향** → 좌우 진동 안정 → 구질 잡힘

⚠️ 한 가지를 잡으면 다른 한 가지를 놓칩니다. 골퍼의 미스 패턴에 따라 정렬 방향을 결정해야 합니다.

**왜 중요한가:** 아이언 7개의 스파인이 제각각 장착되어 있다면 정상적인 퍼포먼스를 장담할 수 없습니다. 같은 방향으로 정렬되어 있어야 같은 스윙 플레인에서 일관된 볼이 나옵니다. **골프는 일관성의 스포츠입니다.**

🆕 최신 트렌드 / 클럽메이커스 공작소 스파인 정렬 작업 가능




**→ 공작소 작업: [헤드 · 샤프트](/workshop#head-shaft)**

### 샤프팅 비드 (Shafting Beads) 🛠️
*Shafting Beads*

### 세트 메이크업 (Set Make-up)
*Set Make-up*

> 14개 클럽의 구성.

R&A·USGA 규정상 라운드에 휴대 가능한 클럽은 최대 14개. 어떤 14개로 채울지 = 세트 메이크업.

**세트 메이크업 검증 항목 (PART 11):**

### 솔 (Sole) 🛠️
*Sole*

> 헤드의 바닥면. 지면과 직접 닿는 부분.

솔의 형태가 디봇 패턴과 임팩트 일관성을 결정합니다. 와이드 솔(넓음)은 디봇 없이 쓸어치는 골퍼에게 유리하고, 내로우 솔(좁음)은 디봇이 깊은 골퍼에게 유리합니다. **시니어·여성 골퍼는 거의 항상 와이드 솔.**

**🔧 솔과 라이각의 관계 — 다이나믹 라이 테스트의 원리**

라이 테이프를 솔에 붙이고 임팩트 보드에서 타격하면, **자국이 남는 솔의 위치**가 라이각 편차를 보여줍니다.

| 자국 위치 | 의미 | 처방 |
|---|---|---|
| 토 쪽 | 라이가 플랫 (헤드가 토부터 닿음) | 업라이트로 1° 조정 |
| 솔 중앙 | 라이가 정확 | 변경 불필요 |
| 힐 쪽 | 라이가 업라이트 | 플랫으로 1° 조정 |

**🔧 솔과 그라인드 — 웨지 작업**

솔 형태가 그라인드 분류를 결정합니다. 와이드 솔은 디봇이 얕은 골퍼에게, 내로우 솔은 디봇이 깊은 골퍼에게.

**현장 사례 (시니어 피팅):** *"시니어 아이언은 와이드 솔 + 두꺼운 탑라인. 머슬백 + 낮은 로프트 + 헤드스피드 감소 = 낮고 짧은 볼. 시니어 골퍼에게 가장 흔한 미스매칭."*

### 스윙웨이트 (SW)
*Swing Weight*

> 클럽을 휘두를 때 느껴지는 헤드 쪽 무게감.

*Spin Loft*

### 얼리 릴리즈 (Early Release / Casting)
*Early Release / Casting*

- **토크**: 낮음 (2.8~3.5°)

### 에폭시 (Epoxy) 🛠️
*Epoxy*

> 헤드와 샤프트를 영구 결합하는 2액형 접착제.

골프 클럽 조립의 핵심 자재. 단순 접착제(Glue)가 아닌 **레진(Resin) + 경화제(Hardener)** 2가지를 정확한 비율로 혼합한 화학 접착제.

**2가지 종류:**

| 타입 | 경화 시간 | 강도 | 용도 |
|---|---|---|---|
| **슬로우셋** | 24시간 | 최고 | 표준 클럽 조립 |
| **패스트셋** | 30~90분 (투어셋) | 표준 | 긴급 작업 |

⚠️ **클럽메이킹은 패스트푸드가 아닙니다.** 슬로우셋이 거의 항상 정답입니다. 패스트셋은 강도 손실이 있어 1년 내 분리될 위험이 있습니다.

**작업 시 주의:**
- 샤프트 팁 표면 처리 필수 (스틸: 연마 / 그라파이트(그래파이트): 코팅 제거)
- 호젤 내부 청결 (오래된 에폭시·기름 제거)
- 적정량 사용 (과도하면 임팩트 시 균열 가능)
- 헐거운 호젤은 **샤프팅 비드 4~6%** 혼합

⚠️ 한 번 굳은 에폭시는 **히트건 가열로만** 분리 가능. 이것이 샤프트 추출 작업의 원리.

→ 리샤프트 / 샤프트 분리 / 샤프팅 비드 항목 참조

### 자체 제품 vs 커스텀 클럽 vs 클럽피팅
*In-house Product vs Custom Club vs Club Fitting*

> 헷갈리기 쉬운 세 가지 개념.

| 분류 | 정의 | 클럽메이커스 사례 |
|---|---|---|
| **자체 제품** | 우리가 직접 설계·제조한 헤드·샤프트 | VERSUS 샤프트 / KRZ 유틸리티 / Define 샤프트 |
| **커스텀 클럽** | 기성 부품을 선택·조합해 1개 골퍼에게 맞춤 제작 | 헤드(타사) + VERSUS 샤프트 + 미드사이즈 그립 |
| **클럽피팅** | 골퍼·신체·스윙을 분석해 적합 스펙 도출 | 풀피팅 4단계 프로세스 |

쉽게:
- **자체 제품** = 우리 브랜드 부품 자체
- **커스텀 클럽** = 부품을 골라 만든 그 골퍼만의 클럽
- **클럽피팅** = 어떤 부품·스펙이 맞는지 결정하는 분석 과정

피팅 → 추천 스펙 도출 → 커스텀 클럽 제작 → (자체 제품 활용 가능) — 이런 흐름이 일반적.

### 카운터밸런스 (Counter-Balance) 🆕
*Counter-Balance / Counter-Weight*

> 그립 끝(버트)에 무게를 추가해 클럽의 밸런스 포인트를 위로 옮긴 설계.

총중량은 늘리면서 스윙웨이트는 낮추거나 유지합니다. 결과: **클럽이 손에 더 가볍게 느껴지면서도 헤드 안정성은 유지됩니다.**

**4가지 효과:**

1. **헤드 스피드 증가** — 클럽이 가볍게 느껴져 더 빠르게 휘두를 수 있음 (단, 추가 총중량을 감당할 수 있을 때)
2. **헤드에 무게 추가 가능** — 헤드를 무겁게 해도 SW가 과도해지지 않음
3. **긴 샤프트의 밸런스 보정** — 길이를 늘려도 휘두름이 가능
4. **임팩트 안정성** — MOI 증가로 스트로크가 흔들리지 않음

**구현 방식:**
- 샤프트 자체 설계 (밸런스 포인트가 그립 쪽) — 예: True Temper HZRDUS Gen4 Red CB
- 버트 웨이트 인서트 (8g~32g 추 삽입)
- 그립 아래 납 테이프

**클럽메이커스 활용:**
- 카운터밸런스 = MOI 매칭의 강력한 도구
- 4g 버트 무게 = 1 SW 포인트 감소 / 2g 팁 무게 = 1.5 SW 포인트 증가
- → 책 PART 4 (9~10장) + REFERENCE R-4

**퍼터 카운터밸런스:**
TaylorMade Spider ZT Counter Balance 등 — 헤드를 395g까지 무겁게 하면서 그립 쪽 무게로 균형. MOI 증가 → 스트로크 안정.

### 카운터밸런스 작업 (Counter-Balance Work) 🛠️
*Counter-Balance Adjustment*

> 그립 끝에 무게를 추가해 SW를 낮추면서 총중량은 유지하는 작업.

**작업 도구:**

### Club Speed (클럽 스피드) 📊
*Club Speed*

> 임팩트 직전 클럽 페이스 중앙의 속도.

거리의 가장 근본적인 원천. **클럽 스피드 1mph = 약 2.5야드 캐리 잠재력**. 일반적으로 한 클럽 짧아질 때마다 약 2mph 감소.

투어 평균:

| 구분 | 드라이버 | 6번 아이언 |
|---|---|---|
| PGA 투어 | 112 mph | 92 mph |
| LPGA 투어 | 94 mph | 78 mph |

### 키네마틱 시퀀스
*Kinematic Sequence*

> 골프 스윙에서 골반·몸통·팔·클럽 4개 분절이 가속·감속하는 순서.

이상적 순서는 트랜지션에서 **1(골반) → 2(몸통) → 3(팔) → 4(클럽)**. 큰 분절이 먼저 시작하고, 작은 분절이 마지막에 최고속도에 도달하는 구조. 채찍 효과와 비슷하지만 더 복잡합니다.

3가지 체크포인트:
1. **테이크어웨이 순서** — 4 → 3/2 → 1 (원위부 먼저)
2. **트랜지션 순서** ⭐ 가장 중요 — 1 → 2 → 3 → 4 (근위부 먼저)
3. **피크 스피드 순서** — 1 → 2 → 3 → 4

K-VEST, TrackMan, Swing Catalyst 등 3D 분석 장비로 측정합니다. 클럽메이커스는 2007년부터 이 측정 결과를 클럽 변수로 변환하는 자체 시스템을 발전시켜 왔습니다.

### 토크 (Torque)
*Torque*

> 샤프트가 비틀리는 정도. 단위는 도(°).

샤프트의 버트를 고정하고 팁에 1파운드 추를 매달았을 때 비틀리는 각도. 일반 샤프트는 2~8° 범위.

| 토크 | 감각 | 적합 골퍼 |
|---|---|---|
| 낮음 (2.8~3.5°) | 단단함 | 빠른 템포·강한 골퍼 |
| 중간 (3.5~4.5°) | 균형 | 일반 |
| 높음 (4.5~5.5°) | 부드러움 | 느린 템포·시니어 |

토크는 결과에 영향이 작지만 **감각에 영향이 큽니다.** 같은 플렉스라도 토크가 1° 다르면 골퍼가 "전혀 다른 샤프트"라고 느낍니다.

⚠️ 퍼터의 "제로토크"는 다른 개념입니다 — 샤프트 비틀림이 아니라 헤드의 회전 토크를 의미합니다.

### 트리밍 (Trimming) 🛠️
*Tip Trim*

> 샤프트의 팁을 잘라내는 작업.

같은 샤프트라도 팁을 1/4인치 더 자르면 강성이 올라갑니다. 트리밍 차트에 따라 클럽별로 다르게 자릅니다.

- 팁 트리밍 증가 → Tip Stiff 방향 → 얼리 릴리즈 억제
- 팁 트리밍 감소 → Tip Soft 방향 → 헤드 복원 가속

여성 골퍼: 남성 표준(½인치)보다 줄여 ¼인치로. **팁 유연성을 유지해 탄도를 확보합니다.**

### Face Angle (페이스 앵글) 📊
*Face Angle*

> 임팩트 순간 페이스가 향하는 방향.

볼의 출발 방향을 결정하는 1차 변수. 드라이버 기준 **출발 방향의 약 80~85%를 결정**합니다.

- **페이스 +3° (열림)** → 우측으로 출발
- **페이스 0° (스퀘어)** → 정면 출발
- **페이스 -3° (닫힘)** → 좌측으로 출발

⚠️ 스트레이트 샷은 페이스 앵글이 0°이지만, 모든 골퍼가 0°를 추구할 필요는 없습니다. **골퍼의 구질에 따라 이상적인 페이스 앵글이 다릅니다.**

골퍼들이 흔히 "패스가 잘못됐다"고 말하지만, 실제로는 페이스가 원인인 경우가 더 많습니다.

### Face to Path 📊
*Face to Path*

> 페이스 앵글과 클럽 패스의 차이. 볼 휨의 직접 원인.

볼이 휘는 구질을 결정하는 핵심 데이터. 드로/훅/페이드/슬라이스를 직접 만드는 변수.

- **+ (플러스)**: 페이스가 패스보다 우측 → 슬라이스 또는 페이드
- **− (마이너스)**: 페이스가 패스보다 좌측 → 훅 또는 드로

| 페이스 | 패스 | Face to Path | 결과 |
|---|---|---|---|
| 0° | 0° | 0 | 스트레이트 |
| 0° | +2° | -2° | 드로 |
| 0° | -2° | +2° | 페이드 |
| +3° | 0° | +3° | 푸시 슬라이스 |
| -2° | -2° | 0° | 풀 (직선 좌측) |

**구체적 휨 예시 (드라이버 캐리 기준):**

| 투어 | Face to Path | 캐리 거리 | 휨 거리 |
|---|---|---|---|
| PGA (275야드) | -2° | — | 19야드 좌측 |
| PGA (275야드) | +5° | — | 44야드 우측 |
| LPGA (218야드) | +2° | — | 14야드 우측 |
| LPGA (218야드) | -5° | — | 32야드 좌측 |

### 페이스 프로그레션 (Face Progression)
*Face Progression*

> 호젤(샤프트가 꽂히는 부분) 중심선에서 페이스 앞면까지의 거리.

현장에서 "오프셋"으로 더 자주 불립니다.

- 큰 오프셋: 페이스가 호젤 뒤쪽에 위치 → 닫히는 시간 확보 → 슬라이스 완화 → 일반·시니어용
- 작은 오프셋: 페이스가 호젤과 가까움 → 컨트롤 우수 → 상급자용

플레이어용 머슬백 아이언은 거의 항상 작은 오프셋, 게임 임프루브먼트 아이언은 큰 오프셋.

### 풀피팅 (Full Fitting)
*Full Fitting*

> TPI 신체평가 + K-VEST 스윙분석 + 클럽 매칭의 4단계 통합 피팅.

클럽메이커스의 핵심 서비스:

1. **신체 평가** — TPI 가동범위·근력 균형
2. **스윙 분석** — K-VEST 키네마틱 시퀀스
3. **데이터 매칭** — MOI · 샤프트 · 헤드 스펙 산정
4. **정밀 제작** — 라이·로프트·길이·SW 미세조정

플로어 피팅(부분 조정)과 다릅니다. 풀피팅은 신체에서 시작해 클럽으로 끝나는 전체 흐름.

### FLO (Flat Line Oscillation) 🆕
*Flat Line Oscillation*

> 샤프트 스파인 측정·정렬의 가장 정확한 방법.

샤프트를 클램프에 고정하고 헤드부에 레이저 포인터를 부착. 샤프트를 튕길 때 레이저가 일직선으로 진동하면 정렬됨, 타원형이면 미정렬.

**측정 방법 비교:**

| 방법 | 정확도 | 장비 |
|---|---|---|
| 베어링 | 보통 | 단순 |
| **FLO** | 높음 | 클램프 + 레이저 |
| CPM 보드 | 높음 | CPM 측정 장비 |

레이저가 일직선으로 진동하는 방향이 스파인의 두 정렬 옵션 중 하나(3-9시 또는 12-6시)입니다. 골퍼의 미스 패턴에 따라 어느 정렬을 선택할지 결정합니다.

🆕 최신 트렌드 / → 샤프트 스파인 항목 참조




**→ 공작소 작업: [헤드 · 샤프트](/workshop#head-shaft)**

*Flex*

⚠️ **제조사마다 기준이 다릅니다.** A 회사의 R이 B 회사의 S와 같을 수 있습니다. 그래서 현대 피팅에서는 플렉스 표기보다 **CPM(실측 진동수)을 우선**합니다.

플렉스만 보고 결정하지 마세요. CPM, EI, 토크를 함께 봐야 합니다.

### 호젤 (Hosel) 🛠️
*Hosel*

> 헤드에서 샤프트가 꽂히는 부분.

호젤의 길이와 위치가 페이스 프로그레션, 라이각, 무게중심에 모두 영향을 줍니다. 호젤이 길수록 무게가 힐 쪽에 치우치고, 호젤이 짧을수록(또는 호젤리스 디자인) 무게가 분산됩니다.

**🔧 호젤이 등장하는 4가지 클럽메이커스 작업:**

**1. 라이각 조정 (Bending)**
호젤을 위·아래로 휘어 라이각 변경. **그루브라인을 지면과 평행**으로 정렬한 후 작업. 포지드는 ±2°, 캐스팅은 ±1° 권장.
→ 라이/로프트 조정 10,000원

**2. 로프트 조정**
호젤을 앞·뒤로 휘어 로프트 변경. 라이각 작업과 같은 밴딩기 사용.
→ 라이/로프트 조정 10,000원

**3. 샤프트 추출 (Pull)**
히트건으로 호젤 가열 → 에폭시 연화 → 샤프트 분리.
→ 헤드·샤프트 분리 10,000원

**4. 호젤 내부 청소 + 재조립**
오래된 에폭시 제거. 새 샤프트 장착 전 필수.

⚠️ **호젤은 반복적으로 휘면 약해집니다.** 피팅 의뢰 시 "이 클럽은 작년에 라이각 1° 업라이트로 조정했다"는 정보를 반드시 알려주세요. 같은 호젤을 다시 휘면 균열 위험이 있습니다.

→ 라이각 조정 작업 / 그루브라인 / 샤프트 분리 항목 참조




**→ 공작소 작업: [헤드 · 샤프트](/workshop#head-shaft)**

### 호젤 사이즈 (Hosel Bore Size)
*Hosel Bore Size*

> 헤드의 호젤 안쪽 지름.

샤프트 팁 사이즈와 정확히 매칭되어야 합니다. **에폭시로 큰 차이를 메꿀 수 없습니다** — 0.335" 샤프트를 0.350" 호젤에 끼우면 결국 분리됩니다.

| 사이즈 | 주 용도 |
|---|---|
| **.335"** | 우드·드라이버·하이브리드 (대부분) |
| **.350"** | 일부 우드·하이브리드 |
| **.355"** | 테이퍼 팁 아이언 (대부분의 스틸) |
| **.370"** | 패러럴 팁 아이언·하이브리드 (그라파이트(그래파이트) 다수) |

미세한 헐거움은 **샤프팅 비드(Shafting Beads)를 4~6%** 비율로 에폭시에 섞어 보정. 그러나 큰 차이는 메꿀 수 없습니다.

→ 페룰(페럴) / 테이퍼 팁 / 패러럴 팁 / 샤프팅 비드 항목 참조

### 허용 범위
*Tolerance Range*

> 스윙 조건이 흔들렸을 때 결과가 벗어나는 최대 범위.

피팅의 핵심 설계 기준. 평균 비거리를 늘리는 것이 아니라, **최악의 샷이 어디까지 벗어나지 않게** 만드는 것이 목표입니다.

| 판단 기준 | 의미 |
|---|---|
| 평균 비거리 | 중간값 (분포를 숨김) |
| 편차 | 결과가 흩어지는 정도 |
| **최악의 샷** | 조건 무너졌을 때의 결과 |
| **허용 범위** | 최악이 벗어나는 최대 범위 |

라운드 스코어를 결정하는 것은 완벽한 샷이 아니라 최악의 샷입니다.

### CPM (Cycles Per Minute)
*Cycles Per Minute*

> 샤프트의 분당 진동 횟수. 강성의 객관적 지표.

플렉스 표기(R/S/X)는 제조사마다 다릅니다. CPM은 실측 수치이므로 객관적 비교가 가능합니다.

| 클럽 | 일반 CPM 범위 |
|---|---|
| 드라이버 R | 230~245 |
| 드라이버 S | 245~260 |
| 드라이버 X | 260+ |
| 7번 아이언 R | 285~295 |
| 7번 아이언 S | 295~310 |

CPM은 **헤드스피드와 매칭**합니다. 90mph 골퍼에게 270 CPM은 너무 강함, 105mph 골퍼에게 240 CPM은 너무 약함.

### EI (Bending Stiffness Profile)
*EI Profile*

> 샤프트의 구간별 강성 분포.

CPM이 한 지점의 강성이라면, EI는 **샤프트 전체에 걸친 강성의 변화**입니다. 어디가 부드럽고 어디가 단단한가.

5가지 유형:
- **Tip Soft**: 버트 강 / 팁 부드러움 → Dynamic Loft 증가, 헤드 복원 빠름
- **Tip Stiff**: 버트 부드러움 / 팁 강 → Dynamic Loft 억제, 얼리 릴리즈 보정
- **Mid (균형형)**: 가장 넓은 적용 범위
- **Butt Soft**: 전환 시 충격 흡수
- **Progressive (역진형)**: 버트~팁 순차 강화. 급전환 골퍼에 이상적

EI 매칭은 키네마틱 시퀀스 패턴과 직접 연결됩니다. → 부록 D EI 매칭 가이드 (책)

### WTF (Wrist-to-Floor)
*Wrist-to-Floor*

> 손목 주름에서 바닥까지의 거리.

신장 대비 팔 길이를 보는 지표. 같은 신장이라도 WTF가 다르면 클럽 길이와 라이각이 달라집니다.

- 5cm 이상 김 → 길이 +0.5", 라이 업라이트

## SEO/AEO 최적화

각 용어 페이지에 schema.org `DefinedTerm` 마크업:

```json
{
  "@context": "https://schema.org",
  "@type": "DefinedTerm",
  "name": "라이 앵글",
  "alternateName": "Lie Angle",
  "description": "클럽이 지면에 놓였을 때 샤프트와 지면 사이의 각도...",
  "inDefinedTermSet": "https://myclubmakers.com/glossary",
  "termCode": "lie-angle"
}
```

전체 사전에는 `DefinedTermSet`. 트랙맨/트렌드/작업 통합 페이지에는 `Article` + `DefinedTerm` 다중 마크업으로 검색 노출 강화.
