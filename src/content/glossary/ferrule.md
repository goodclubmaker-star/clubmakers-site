> 헤드와 샤프트의 연결부에 끼우는 작은 플라스틱·고무 부품.

표기 주의: 한국에서는 **"페룰"과 "페럴"** 두 표기가 혼용됩니다. 영문 ferrule의 한글 표기 차이일 뿐 같은 부품입니다.

기능:
- 임팩트 충격 흡수
- 외관 마무리 (헤드~샤프트 전환부)
- 호젤 보호

**호젤 사이즈와 페룰(페럴) 매칭:**

| 호젤 사이즈 | 페룰(페럴) / 샤프트 팁 |
|---|---|
| **.335"** | 우드·드라이버 |
| **.350"** | 일부 우드·하이브리드 |
| **.355"** | 테이퍼 팁 (아이언, 대부분의 스틸) |
| **.370"** | 패러럴 팁 (아이언·하이브리드, 그라파이트(그래파이트) 다수) |

**현장 사례 (작업시트):** *"NS Pro 950 Neo S 5번 샤프트를 6번에 장착, 0.355 사이즈 페럴 사용"* — 페룰(페럴) 사이즈는 작업 시 항상 명시됩니다.

⚠️ 헐거운 페룰(페럴)은 임팩트 시 미세한 흔들림과 잡음을 만듭니다. 즉시 교체 권장.

**클럽메이커스 공임표:**
- **페룰(페럴) 헐거움 5,000원** (조임 작업)
- **페룰(페럴) 교체 18,000원** (헤드 분리 → 샤프트 추출 → 페룰 교체 → 재조립 작업 포함)
